**Usage Intructions:**

## .env file : Put .env file at the root of frontend directory
```bash
NEXT_PUBLIC_API_URL = http://localhost:4000
```

## Install dependencies
```bash
npm install
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
